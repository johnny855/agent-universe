## 📋 需要您在电脑上完成的操作

### 步骤 1：获取 Google 验证码（最重要）

1. 在电脑浏览器打开：**https://search.google.com/search-console**
2. 登录您的 Google 账号
3. 点击 **"添加资源"**
4. 输入：`https://johnny855.github.io/agent-universe/`
5. 选择验证方式：**HTML 标签**（点击展开查看）
6. 复制 `<meta name="google-site-verification" content="xxx" />` 中的 **content 值**（约20个字符）
7. 将验证码发送给 AI 助手，格式：
   ```
   我的 Google 验证码是：xxxxxxxxxxxxxxxxxxxx
   ```

### 步骤 2：获取 Bing 验证码（可选但推荐）

1. 在电脑浏览器打开：**https://www.bing.com/webmasters**
2. 使用 Microsoft 账号登录
3. 点击 **"添加站点"**
4. 输入：`https://johnny855.github.io/agent-universe/`
5. 选择验证方式：选择 **"HTML 标签验证"**
6. 复制验证码，格式：
   ```
   我的 Bing 验证码是：xxxxxxxxxxxxxxxx
   ```

### 步骤 3：AI 助手将自动完成

收到验证码后，AI 助手会：
1. 替换代码中的占位符
2. 构建网站
3. 部署到 gh-pages 分支
4. 在各平台完成验证和 sitemap 提交
